# Asset Sync Tool

A Unity Editor extension for synchronizing assets between projects and external sources.

## Features

- Sync assets between multiple Unity projects
- External asset source integration
- Automated asset management
- Custom sync rules and filters

## Installation

1. Open Unity Package Manager
2. Click the "+" button
3. Select "Add package from git URL"
4. Enter the repository URL or local path

## Usage

- Access the Asset Sync window through `Window > Asset Sync Tool`
- Configure sync settings in the Asset Sync Manager
- Use the menu items for quick sync operations

## Requirements

- Unity 2020.3 or higher
- No additional dependencies
